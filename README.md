# Natuurkunde-PO-Morsecode-For-Dummies

In dit onderzoek wordt onderzocht hoe morsecode vertaald kan worden met behulp van een signaal. Het bekendste signaal om morsecode door te geven is geluid, maar dit is zeker niet de enige mogelijkheid, er zijn namelijk meerdere manieren. Zo kun je ook gebruikmaken van licht. Een ledlampje dat aan en uit kan gaan is daarvoor al voldoende.

Deze github is gemaakt om alle codes te delen.
